# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in MarketsCoupons, please report it responsibly.

**Contact:** evertonmiranda777@gmail.com

**What to include:**
- Description of the vulnerability and impact
- Steps to reproduce
- Affected URLs/endpoints
- Your contact info (so we can follow up)

**What we promise:**
- Acknowledgement within 72 hours
- Status update within 7 days
- Public credit (unless you prefer to stay anonymous)

**Please do NOT:**
- Publicly disclose the vulnerability before we've had a chance to fix it
- Run automated scanners that generate excessive load (DoS-equivalent)
- Access, modify, or delete data that isn't yours
- Attempt to deanonymize other users

## Scope

In scope:
- `marketscoupons.com` and `*.marketscoupons.com`
- `marketscoupons.vercel.app`
- API endpoints under `/api/*`

Out of scope:
- Third-party services we link to (prop firms, Brevo, Resend, Supabase, Vercel, Cloudflare)
- Social engineering of staff or users
- Physical attacks
- Findings from automated scanners with no proof of impact

## Supported Versions

Only the latest deployed production version (`main` branch on `marketscoupons.com`) is supported and patched. Older deployments are not maintained.

## Hall of Fame

Security researchers who report valid vulnerabilities will be acknowledged here (with permission).
